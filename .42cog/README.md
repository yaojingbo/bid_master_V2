# .42cog/ - 认知敏捷法文档

本目录采用 RCSW 工作流：Real → Cog → Spec → Work

## 子目录

- `meta/` - 项目元数据（名称、定位、价值、技术栈、里程碑）
- `real/` - 现实约束（安全规则、合规要求、技术债务、已知风险）
- `cog/` - 认知模型（实体关系、领域概念）
- `work/` - 工作记录（任务分解、进度追踪）
- `others/` - 其他文档
- `spec/` - 规约模板

## 工作流

1. **Real** - 分析现实约束和需求
2. **Cog** - 建立认知模型
3. **Spec** - 输出规约
4. **Work** - 执行开发

详见各子目录 README.md
