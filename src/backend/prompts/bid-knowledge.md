> 整合招标文件要素提取和模拟编制两个技能的领域知识，消除重复。
> 涵盖资质体系、评标方法、基准价计算、业绩要求、定标方式、区域规律。

## 一、资质等级体系

### 1.1 设计类资质等级链

```
工程设计综合甲级（最高，通吃全部）
    ↓
工程设计行业甲级（如：市政工程设计行业甲级）
    ↓
工程设计行业（燃气、轨道除外）甲级
    ↓
工程专业设计甲级（如：市政行业排水专业甲级）
    ↓
工程设计行业乙级
    ↓
工程专业设计乙级
```

### 1.2 设计类资质-规模对照表

| 项目规模 | 推荐资质等级 | 常见配置 |
|----------|-------------|----------|
| **大型**（≥10万吨/日 或 投资≥5亿） | 综合甲级 或 行业甲级 四选一 | ①综合甲级 ②市政行业甲级 ③市政（燃气轨道除外）甲级 ④排水专业甲级 |
| **中型**（2-10万吨/日） | 行业甲级 或 专业甲级 | ①综合甲级 ②市政行业甲级 ③排水专业甲级 |
| **小型**（<2万吨/日） | 行业乙级及以上 | ①综合甲级 ②市政行业乙级及以上 |
| **特殊（含化工）** | 环境工程专项甲级+建筑乙级 | ①综合甲级 ②环境工程（水污染防治）甲级+建筑行业乙级 |

**设计类项目负责人资格**：

| 项目类型 | 项目负责人要求 |
|----------|---------------|
| 市政给排水 | 注册公用设备工程师（给水排水） |
| 建筑 | 一级注册建筑师 |
| 结构 | 一级注册结构工程师 |
| 环保 | 注册环保工程师 |

**设计类资质选择规律**：
- **综合甲级**通吃全部设计项目，是最大通行证
- **市区大型项目**通常四选一
- **工业污水项目**可能要求环境工程专项甲级
- **小型项目**可降低至市政行业乙级
- 设计类项目**一般不接受联合体投标**

### 1.3 施工总承包资质

**等级链**：特级 > 一级 > 二级 > 三级

**常见资质类型（市政/环保类）**：

| 资质类型 | 适用工程 | 常见等级要求 |
|---------|---------|------------|
| 市政公用工程施工总承包 | 市政道路、管网、水厂 | 一级或二级 |
| 环保工程专业承包 | 污水处理、垃圾处理 | 一级或二级 |
| 建筑工程施工总承包 | 房建类工程 | 一级或二级 |
| 机电工程施工总承包 | 机电安装工程 | 一级或二级 |
| 水利水电工程施工总承包 | 水库、堤防、灌溉 | 一级或二级 |

**施工类资质-规模对照表**：

| 项目类型 | 推荐资质配置 | 常见组合 |
|----------|-------------|----------|
| **市政施工+O（PC+O）** | 市政施工总承包三级及以上 + 机电施工/安装一级 | 双资质组合 |
| **纯土建施工** | 市政施工总承包二级及以上 | 单资质 |
| **机电设备改造** | 机电施工总承包一级 | 单资质 |
| **综合施工（含设备）** | 市政施工总承包一级 + 机电安装一级 | 双资质 |

**施工类项目负责人**：

| 项目类型 | 建造师要求 | 专业 |
|----------|-----------|------|
| 市政工程 | 一级注册建造师 | 市政公用工程 |
| 机电安装 | 一级注册建造师 | 机电工程 |
| 建筑工程 | 一/二级注册建造师 | 建筑工程 |

**施工业绩门槛对照**：

| 项目投资规模 | 业绩金额门槛 | 门槛比 |
|-------------|-------------|--------|
| ≥5000万元 | 2000-5000万元 | 40%-100% |
| 1000-5000万元 | 500-1500万元 | 30%-50% |
| <1000万元 | 按类型设定 | - |

### 1.4 设备类资质/条件对照

| 要求项 | 常见配置 |
|--------|----------|
| 资质 | 制造商 或 授权代理商 |
| 安全生产许可证 | 如涉及安装工程则需 |
| 联合体 | 允许（安装单位牵头+设备供应商） |

**三类项目核心区别**：

| 维度 | 设计类 | 施工类 | 设备类 |
|------|--------|--------|--------|
| **资质体系** | 设计资质 | 施工总承包资质 | 供应/安装资质 |
| **项目负责人** | 注册工程师 | 注册建造师 | 项目经理（不一定要求注册） |
| **安全生产** | 一般不要求 | 必须有许可证 | 如含安装则需 |
| **业绩类型** | 设计业绩 | 施工业绩 | 设备供应/安装业绩 |
| **保证金** | 合同价2% | 合同价5-10% | 合同价5% |
| **预付款** | 0-10% | 10-30% | 30%（最高） |

### 1.5 资质组合要求

招标文件中常见的资质组合模式：
- **单项资质**：仅需一种资质，如"市政公用工程施工总承包二级及以上"
- **双项资质**：需两种资质同时满足，如"市政二级 + 环保二级"
- **主+增项**：主资质必须，增项资质可联合体满足
- **联合体**：允许两家企业联合投标，联合体各方均需满足各自对应的资质

### 1.6 联合体设置规律

| 项目类型 | 联合体策略 | 牵头人要求 |
|----------|-----------|-----------|
| **设计类** | 通常不接受 | - |
| **施工+O** | 接受，≤3家 | 主体施工单位（市政资质） |
| **设备采购安装** | 接受，≤2家 | 设备安装单位 |
| **纯施工** | 视项目而定 | 资质等级最高的单位 |

### 1.7 动态核查

浙江省特色要求：
- 投标时资质需通过动态核查
- 动态核查不通过的，投标无效
- 核查内容包括：注册人员、技术装备、工程业绩等

## 二、评标方法分类

### 2.1 浙江省三类项目评分权重总表

| 项目类型 | 资信标 | 技术标 | 商务标 | 决胜因素 |
|----------|--------|--------|--------|----------|
| **设计类** | 5-10分 | 80-85分 | 10分 | 技术方案质量 |
| **施工类** | 10-20分 | 30-50分 | 30-50分 | 综合评分（视方法而定） |
| **设备类** | 5分 | 25分 | 70分 | 设备报价竞争力 |

### 2.2 评定分离（浙江省设计类项目标配）

**定义**：评标委员会负责评审打分推荐候选人，定标委员会另行组建负责最终确定中标人。

**流程**：
1. 评标委员会评审 → 推荐中标候选人（通常3-5名）
2. 定标委员会另行召开定标会议
3. 票决法/集体议事法确定中标人

**依据**：浙发改公管〔2023〕256号《浙江省工程建设项目招标投标"评定分离"操作指引（试行）》

**识别标志**：
- 招标文件前附表注明"☑采用评定分离"
- 定标章节标题含"适用于'评定分离'项目"
- 设计类项目几乎100%采用

### 2.3 综合评估法（最常见）

**原理**：将评标因素量化为分数，按总分排名。

**分值结构**：`总分 100 分 = 资信标 X 分 + 技术标 X 分 + 商务标 X 分`

### 2.4 设计类项目评分规律

**资信标评分项（5-10分）**：

| 评分项 | 常见分值 | 评分标准 |
|--------|----------|----------|
| **企业业绩** | 2-4分 | 单个规模达标业绩得分，最高2-4分 |
| **项目负责人业绩** | 1-2分 | 同企业业绩要求，但针对个人 |
| **企业荣誉** | 0-4分 | 省级/国家级勘察设计奖（部分项目无此项） |
| **人员力量** | 2-6分 | 注册+高工组合，每人0.5-1分 |
| **投标人资质** | 0-2分 | 环境工程专项甲级等特殊资质加分 |

**技术标评分项（80-85分）**：

| 评分项 | 常见分值范围 | 说明 |
|--------|-------------|------|
| 项目理解/现状调查 | 5-15分 | 对项目背景、现状的分析深度 |
| **设计方案** | 30-48分 | **核心评分项**，占技术标37%-75% |
| 重点难点 | 5-10分 | 运营安全、高浓度冲击等 |
| 设备选型 | 5-10分 | 设备方案合理性 |
| 投资估算/限额设计 | 2-5分 | 造价控制能力 |
| 进度计划 | 2-5分 | 设计周期安排 |
| 后续服务 | 2-5分 | 现场服务、竣工图配合 |
| 方案效果图 | 0-10分 | 部分项目要求 |

**技术标评审方式 — 两阶段评审法**（所有设计类项目共性）：
1. 第一阶段：记名投票定档（第一档极佳/第二档优良/第三档极差）
2. 第二阶段：在档位对应分值区间内评分

| 档位 | 单项评分范围 |
|------|------------|
| 第一档（极佳） | 分值上限的100%~60% |
| 第二档（优良） | 分值上限的85%~60% |
| 第三档（极差） | 分值上限的85%~20% |

- 调档需评标委员会2/3以上成员同意
- 有效评分>5个时去掉最高最低后取平均值

**商务标评分（10分）**：

| 参数 | 常见配置 |
|------|----------|
| 基准价方法 | 随机抽取法（均等权重平均法 / 技术排序平均法） |
| E1（高于扣分） | 0.2分/1% |
| E2（低于扣分） | 0.1分/1% |
| K值范围 | 5-20（随机抽取） |

### 2.5 施工类项目评分规律

**资信标评分项（10-20分）**：

| 评分项 | 常见分值 | 评分标准 |
|--------|----------|----------|
| **信用评价分** | 3-5分 | 浙江省建筑市场监管系统企业信用等级 |
| **企业施工业绩** | 2-5分 | 同类型施工合同业绩 |
| **项目负责人业绩** | 2-3分 | 同类型项目负责人业绩 |
| **项目负责人信用** | 1-2分 | 人员信用评价 |
| **安全生产标准化** | 1-2分 | 安全生产标准化证书 |
| **认证/荣誉** | 1-2分 | ISO认证、工程质量奖等 |

**评标方法（两种主流）**：

| 方法 | 说明 | 适用场景 |
|------|------|---------|
| 技术标通过制 | 技术标只做通过/不通过，不参与排序 | 标准化程度高的项目 |
| 技术标打分制 | 技术标参与评分 | 技术复杂度较高的项目 |

**商务标评分**：

| 参数 | 常见配置 |
|------|----------|
| 基准价 | 最高投标限价×（1-下浮值D%），D随机抽取 |
| 下浮值D范围 | 通常 5%-20% |
| E1（高于扣分） | 2分/1% |
| E2（低于扣分） | 1分/1% |
| 评审区间 | 信用评价排序后取前20名 |

### 2.6 设备类项目评分规律

**资信标评分项（5分，固定配置）**：

| 评分项 | 分值 | 评分标准 |
|--------|------|----------|
| **技术条款响应** | 3分 | 完全响应得3分，否则不得分 |
| **绿色低碳建议** | 1分 | 提供具体方案得1分 |
| **项目组成员** | 1分 | 高工0.5分+注安师0.5分 |

**技术标评分项（25分，暗标评审）**：

| 评分项 | 常见分值 | 说明 |
|--------|----------|------|
| **总体方案** | 6分 | 供货安排、衔接方案、施工组织 |
| **保障措施** | 2分 | 安全保障、不停产措施 |
| **各系统设备方案** | 8-12分 | 按工艺分段评分 |
| **售后服务** | 1-2分 | 服务方案、星级认证 |

**评审方式**：单阶段评分，70%-100%区间打分（非两阶段）

**商务标评分（70分）**：

| 参数 | 常见配置 |
|------|----------|
| 基准价方法 | 随机抽取（二次平均法 / 动态二次平均法） |
| E1（高于扣分） | 2分/1% |
| E2（低于扣分） | 1分/1% |

**关键差异**：设备类商务标权重是设计类的7倍，报价策略对中标影响极大。

### 2.7 其他评标方法

**经评审的最低投标价法**：满足所有实质性要求后，报价最低者中标。适用于通用技术、性能标准的项目。

**票决法**：评标委员会成员一人一票，得票最多者中标。适用于简单小型工程。

**抽签法**：所有符合条件的投标人抽签决定中标人。适用于小型简单工程。

## 三、基准价计算详解

### 3.1 常见基准价公式

#### 公式一：有效报价平均值

```
基准价 = 所有有效报价的算术平均值
变体：去掉N个最高价和N个最低价后的算术平均值
变体：加权平均值（按报价大小赋权）
```

#### 公式二：控制价 × 系数

```
基准价 = 招标控制价 × (1 - 合理下浮率)
下浮率：招标人直接给定 / 所有有效报价下浮率的平均值 / 专家评审确定
```

#### 公式三：综合基准价

```
基准价 = (有效报价平均值 × A% + 招标控制价 × B%) / 100
常见比例：A=70, B=30 或 A=60, B=40
```

### 3.2 基准价方法对照表

| 方法名称 | 计算公式 | 适用项目类型 |
|----------|----------|-------------|
| **均等权重平均法** | 去除极端值后取算术平均 | 设计类 |
| **技术排序平均法** | 按技术分排序取前5名报价平均 | 设计类 |
| **基准值合成法** | 最高限价×（1-K%），K随机 | 设计类 |
| **下浮值法** | 最高限价×（1-D%），D随机抽取 | 施工类 |
| **二次平均法** | 先平均，再与次低价平均 | 设备类 |
| **动态二次平均法** | 按资信+技术排序筛选后计算 | 设备类 |

### 3.3 报价得分计算

**最常见公式**：
```
偏差率 X = (投标报价 - 基准价) / 基准价 × 100%

如果 X > 0（高于基准价）：商务标得分 = 满分 - X × 扣分系数1
如果 X ≤ 0（低于基准价）：商务标得分 = 满分 - |X| × 扣分系数2

常见扣分系数：
  - 正偏差每 1% 扣 1-2 分
  - 负偏差每 1% 扣 0.5-1 分
  - 部分项目负偏差不扣分
```

**注意**：
- 超过有效范围的报价直接废标（如超过控制价）
- 注意"下浮率"的含义：控制价 × (1 - 下浮率)
- 部分项目对低于成本价的报价有否决条款

## 四、业绩证明材料要求

### 4.1 标准业绩证明材料

| 材料 | 说明 | 设计类 | 施工类 |
|------|------|--------|--------|
| 中标通知书 | 招标人发出的中标确认 | 通常必需 | 通常必需 |
| 设计合同/施工合同 | 双方签订的合同 | 通常必需 | 通常必需 |
| 图审合格书 | 施工图审查合格证明 | **设计类核心材料** | 通常不要求 |
| 初步设计批复 | 发改部门批复文件 | 部分项目要求 | 不要求 |
| 竣工验收证书 | 五方签字盖章 | 部分替代图审合格书 | 通常必需 |
| 招标公告网页截图 | 带网址的招标公告 | **市区项目常见要求** | 少见 |
| 业主证明 | 业主盖章的证明文件 | 材料不全时的补充 | 少见 |

### 4.2 业绩认定要点

- **时间认定**：设计类以合同签订时间或图审合格书落款时间为准；施工类以竣工验收日期为准
- **规模认定**：设计类以日处理规模为准；施工类以合同金额或建筑面积为准。注意1万吨/日 ≈ 1万m³/d
- **类型限定**：工业污水项目必须提供工业污水处理厂业绩（不接受生活污水业绩）；注意"新建"和"改（扩）建"是否都接受
- **联合体业绩**：需提供联合体协议明确体现自身承担的工作内容
- **不重复计分**：同一业绩在企业业绩与项目负责人业绩之间只计最高分

### 4.3 业绩规模门槛规律

**规模门槛 = 项目自身规模 × 50%-100%**

| 项目规模等级 | 门槛比例 | 示例 |
|-------------|----------|------|
| 大型（≥5万吨/日） | 80%-100% | 10万吨/日项目 → 门槛8-10万吨/日 |
| 中型（1-5万吨/日） | 60%-100% | 2万吨/日项目 → 门槛1.2-2万吨/日 |
| 小型（<1万吨/日） | 50%-80% | 5000吨/日项目 → 门槛2500-4000吨/日 |

→ **规律**：门槛通常为项目规模的50%-100%，不会出现小项目要大业绩的情况

### 4.4 时间窗口

| 项目规模 | 业绩追溯期 |
|----------|-----------|
| 大型 | 5年（2020.1.1起） |
| 中型 | 4年（2021.1.1起） |
| 小型 | 3年（2022.1.1起） |

### 4.5 证明材料严格度分级

| 严格等级 | 要求材料 | 典型项目 |
|----------|----------|----------|
| 基础级 | 合同+批复/图审 | 小型项目 |
| 标准级 | 中标通知书+合同+图审 | 中型项目 |
| 严格级 | 招标公告+中标+合同+图审 | 大型/市区项目 |

### 4.6 不重复计分原则

- 企业业绩与项目负责人业绩同一项目不重复，只计最高分
- 加分业绩与资格业绩不重复
- 联合体业绩须明确体现自身承担内容

## 五、定标方式规律

| 定标方式 | 适用场景 | 说明 |
|----------|----------|------|
| **票决法** | 浙江省主流 | 投票取最多且过半数 |
| **直接票决法** | 候选人≤5名 | 一次投票定结果 |
| **逐轮票决法** | 候选人>5名 | 多轮淘汰 |
| **集体议事法** | 个别项目 | 组长最终决定 |

**浙江省规律**：评定分离 + 票决法是标准配置。

## 六、地市差异速查

### 衢州市
- 设计类：全部技术标打分制 + 评定分离 + 票决法
- 设备类：综合评估法（5+25+70）+ 评定分离
- 特点：技术标暗标评审，封面统一格式

### 台州市
- 施工类：技术标通过制/打分制 + 评定分离
- 三种方法：资信商务评估法/票决法/综合评估法
- 特点：评审区间机制（信用评价排序取前20名）

### 杭州市
- 设计类：技术标权重偏高（82-85分）
- 业绩要求较严（4件证明材料）
- 特点：限额设计评审要求

## 七、浙江省特色规定

### 7.1 信用评价
- AAA、AA、A、B、C 五个等级
- 信用评价结果可能影响资信标评分
- 部分项目要求信用等级不低于 A 级

### 7.2 动态核查
- 投标前须通过资质动态核查
- 核查不通过视为不具备投标资格
- 核查内容包括注册人员数量、技术装备等

### 7.3 电子招投标
- 浙江省推行全流程电子招投标
- 投标文件需通过电子招投标平台提交
- 注意 CA 证书和电子签章要求

## 八、合同计价方式

| 计价方式 | 说明 | 风险分配 | 适用场景 |
|---------|------|---------|---------|
| 固定总价 | 合同总价不变，除非变更 | 承包方承担价格风险 | 小型、图纸完善的工程 |
| 固定单价 | 单价固定，工程量按实结算 | 发包方承担工程量风险 | 工程量不确定的工程 |
| 可调价格 | 根据约定条件调整价格 | 双方分担风险 | 工期长的工程 |
| 成本加酬金 | 实际成本 + 约定酬金 | 发包方承担主要风险 | 紧急工程、抢险工程 |
