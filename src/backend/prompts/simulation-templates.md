# 招标文件模拟编制模板集

> 三类项目（设计/施工/设备）的完整招标文件模板骨架。

## 一、设计类项目模板

**适用场景**：方案设计、初步设计、施工图设计招标

### 1.1 招标公告

```markdown
# {{项目名称}}设计招标公告

## 1. 招标条件
{{项目名称}}已由{{审批机关}}以{{审批文号}}批准建设，建设资金来自{{资金来源}}，出资比例为{{出资比例}}，项目业主为{{项目业主}}，招标人为{{招标人}}，委托代理机构为{{代理机构}}。项目已具备招标条件，现对该项目的设计进行公开招标。

## 2. 项目概况与招标范围
2.1 建设地点：{{建设地点}}
2.2 投资估算（或概算）：本项目工程费用估算总投资约{{投资估算}}万元。
2.3 建设规模：{{建设规模描述}}
2.4 招标范围：{{招标范围描述}}。包含方案设计（含估算）、初步设计（含概算编制）、施工图设计及后续服务。
2.5 勘察设计服务期限：{{设计周期}}日历天。
  - 方案设计：{{方案天数}}日历天
  - 初步设计：{{初设天数}}日历天
  - 施工图设计：{{施工图天数}}日历天

## 3. 投标人资格要求
3.1 具备建设主管部门颁发的以下设计资质条件之一：
  ① 工程设计综合甲级资质；
  ② {{市政工程设计行业甲级资质}}；
  ③ {{其他可选资质}}。

3.2 本次招标 {{是否接受联合体}} 联合体投标。

3.3 拟派项目负责人须具备 {{注册公用设备工程师（给水排水）}} 证书。

3.4 {{业绩要求描述}}（如有）。

## 4. 招标方式
4.1 公开招标。
4.2 ☑采用评定分离。

## 5-9. 招标文件获取/递交/联系方式
（标准模板，与区域惯例一致）
```

### 1.2 评标办法（设计类标准配置）

```markdown
## 评标办法（技术标打分制综合评估法）

### 分值构成（总分100分）
- A 技术标部分：{{技术分值}}分（≥70分）  ← 区域规律：通常 80-85
- B 资信标部分：{{资信分值}}分（≤20分）  ← 区域规律：通常 5-10
- C 商务标部分：{{商务分值}}分（≤10分）  ← 区域规律：通常 10

### 技术标评审（两阶段法）
第一阶段：记名投票分三档
  - 第一档（极佳）：100%~60%
  - 第二档（优良）：85%~60%（默认）
  - 第三档（极差）：85%~20%
第二阶段：在档位区间内评分

### 技术标评分项（设计类）
| 序号 | 评分项目 | 分值 |
|------|----------|------|
| 1 | 项目解读/现状调查 | {{5-15}} |
| 2 | 设计方案 | {{30-48}} |
| 3 | 重点难点分析 | {{5-10}} |
| 4 | 设备选型 | {{5-10}} |
| 5 | 投资估算/限额设计 | {{2-5}} |
| 6 | 进度计划 | {{2-5}} |
| 7 | 后续服务 | {{2-5}} |
| 8 | 方案效果图（如有） | {{0-10}} |

### 资信标评审
| 评分项 | 分值 | 评分标准 |
|--------|------|----------|
| 企业业绩 | {{2-4}} | {{业绩规模门槛}}的设计业绩 |
| 项目负责人业绩 | {{1-2}} | 同上 |
| 人员力量 | {{2-6}} | 注册+高工组合，每人0.5-1分 |
| 企业荣誉（如有） | {{0-4}} | 省级/国家级勘察设计奖 |

### 商务标评审
- 基准价方法：☑随机抽取法（均等权重平均法/技术排序平均法）
- 扣分公式：高于每1%扣{{E1}}分，低于每1%扣{{E2}}分
- 区域规律：E1=0.2, E2=0.1（浙江省设计类常见配置）
```

### 1.3 合同条款（设计类）

```markdown
## 主要合同条款
- 预付款：{{0% 或 10%}}
- 进度款支付节点：
  1. 初步设计审查通过后 → 30%
  2. 施工图通过审查后 → 50%-80%
  3. 竣工验收完成后 → 85%-95%
  4. 余款在质保期/审计后付清
- 履约保证金：合同总价的 2%
- 违约金：签约合同价的 30%
```

## 二、施工类模板

**适用场景**：施工总承包、EPC、施工+运营（PC+O）等

### 2.1 招标公告

```markdown
# {{项目名称}}{{施工总承包/施工总承包+O}}招标公告

## 2. 项目概况与招标范围
2.1 项目概况：本项目投资估算{{投资估算}}万元，工程概算{{工程概算}}万元，其中建安工程造价{{建安工程费}}万元。
2.2 招标范围：施工图纸所包含的{{施工内容}}以及{{后续调试运营（如有）}}。
2.3 施工总工期：不超过{{工期}}个日历天。
2.4 运营期限：{{运营年限（如有）}}。

## 3. 投标人资格要求
3.1 同时具备以下资质：
  ① {{市政公用工程施工总承包X级}}及以上资质；
  ② {{机电工程施工总承包X级 或 建筑机电安装工程专业承包X级}}（如有）。

3.2 具备有效的企业安全生产许可证。

3.3 业绩要求：
  ① 施工业绩：承接过金额在{{门槛金额}}万元及以上的{{项目类型}}施工业绩；
  ② 运营业绩（如有）：承接过日处理量{{门槛规模}}吨以上的运营业绩。

3.4 拟派项目负责人：{{注册建造师专业}}X级资格。
```

### 2.2 评标办法（施工类标准配置）

```markdown
## 评标办法（技术标通过制综合评估法 或 技术标打分制综合评估法）

### 分值构成（总分100分）——按区域规律选择
方案A（通过制）：资信{{60-70}}分 + 商务{{30-40}}分，技术标通过/不通过
方案B（打分制）：技术{{30-50}}分 + 资信{{10-20}}分 + 商务{{30-50}}分

### 资信标评审（施工类常见配置）
| 评分项 | 分值 | 说明 |
|--------|------|------|
| 信用评价分 | 3-5 | 浙江省建筑市场监管系统信用等级 |
| 企业业绩 | 2-5 | 同类型施工业绩 |
| 项目负责人业绩 | 2-3 | 同类型施工负责人业绩 |
| 项目负责人信用 | 1-2 | 人员信用评价 |
| 认证/荣誉 | 1-2 | 质量体系认证、安全生产标准化 |

### 商务标评审
- 基准价：最高投标限价×（1-下浮值D%），D随机抽取
- 下浮值范围：{{5-20}}（区域规律决定）
- 扣分公式：高于每1%扣{{E1}}分，低于每1%扣{{E2}}分
- 区域规律：E1=2, E2=1（常见不对称扣分）
```

## 三、设备类模板

**适用场景**：设备采购安装、设备改造等

### 3.1 招标公告

```markdown
# {{项目名称}}设备采购安装招标公告

## 2. 招标范围
本项目属于工程货物招标，招标范围包含但不限于{{设备描述}}全部工艺设备、电气自控设备、通用设备及非标设备的采购、安装、调试、试运行及相关服务。

采购安装范围包括：
（1）设备采购、运输、保管、安装及调试；
（2）安装过程中涉及的土建配合、管道连接、电气接线等；
（3）工厂验收（FAT）、现场验收（SAT）、调试及试运行；
（4）操作维护人员培训；
（5）质保期内的维修保养服务；
（6）备品备件供应（不少于2年运行用量）。

招标范围内的造价约{{设备造价}}万元。
```

### 3.2 评标办法（设备类标准配置）

```markdown
## 评标办法（综合评估法）

### 分值构成（总分100分）
- 资信标：5分
- 技术标：25分
- 商务标：70分

### 资信标评审（5分）
| 序号 | 编制内容 | 分值 | 评分细则 |
|------|----------|------|----------|
| 1 | 技术条款响应及创新措施 | 4 | 完全响应得3分；绿色低碳建议得1分 |
| 2 | 项目组成员 | 1 | 高级工程师0.5分 + 注册安全工程师0.5分 |

### 技术标评审（25分，暗标，70%-100%区间打分）
| 序号 | 评分项目 | 分值 |
|------|----------|------|
| 1 | 总体方案 | 6 |
| 2 | 保障措施 | 2 |
| 3 | 主要系统设备方案（按工艺分段） | 8-12 |
| 4 | 电气自控系统方案 | 2-4 |
| 5 | 设备品牌及质量控制 | 1-2 |
| 6 | 售后服务保障 | 1-2 |

### 商务标评审（70分）
- 方法：随机抽取确定（二次平均法 / 动态二次平均法）
- 扣分：高于基准价每1%扣2分，低于每1%扣1分
```

### 3.3 合同条款（设备类）

```markdown
## 主要合同条款
- 预付款：签约合同价的 30%（设备类最高）
- 进度款：按月支付至确认工程量价款的 77%
- 未安装设备：按工程量价款的50%计入次月进度款
- 预付款扣回：从第一次进度款开始每次扣10%
- 竣工结算：验收后支付至85%，试运行达标后95%，余5%质保金
- 履约保证金：合同总价的 5%
- 质保期：{{24个月，关键设备36个月}}
- 备品备件：不少于2年运行用量
- 违约金：合同价30%
```
