# 资质体系对照表

> 按项目类型和规模匹配资质等级，供 Step 4 模拟编制时选择合理的资质要求。

---

## 一、设计类资质等级体系

### 1.1 资质等级链

```
工程设计综合甲级（最高，通吃全部）
    ↓
工程设计行业甲级（如：市政工程设计行业甲级）
    ↓
工程设计行业（燃气、轨道除外）甲级
    ↓
工程专业设计甲级（如：市政行业排水专业甲级）
    ↓
工程设计行业乙级
    ↓
工程专业设计乙级
```

### 1.2 设计类资质-规模对照表

| 项目规模 | 推荐资质等级 | 常见配置 |
|----------|-------------|----------|
| **大型**（≥10万吨/日 或 投资≥5亿） | 综合甲级 或 行业甲级 四选一 | ①综合甲级 ②市政行业甲级 ③市政（燃气轨道除外）甲级 ④排水专业甲级 |
| **中型**（2-10万吨/日） | 行业甲级 或 专业甲级 | ①综合甲级 ②市政行业甲级 ③排水专业甲级 |
| **小型**（<2万吨/日） | 行业乙级及以上 | ①综合甲级 ②市政行业乙级及以上 |
| **特殊（含化工）** | 环境工程专项甲级+建筑乙级 | ①综合甲级 ②环境工程（水污染防治）甲级+建筑行业乙级 |

### 1.3 设计类项目负责人资格

| 项目类型 | 项目负责人要求 |
|----------|---------------|
| 市政给排水 | 注册公用设备工程师（给水排水） |
| 建筑 | 一级注册建筑师 |
| 结构 | 一级注册结构工程师 |
| 环保 | 注册环保工程师 |

## 二、施工类资质等级体系

### 2.1 施工总承包资质等级链

```
施工总承包特级（最高）
    ↓
施工总承包一级
    ↓
施工总承包二级
    ↓
施工总承包三级
```

### 2.2 施工类资质-规模对照表

| 项目类型 | 推荐资质配置 | 常见组合 |
|----------|-------------|----------|
| **市政施工+O（PC+O）** | 市政施工总承包三级及以上 + 机电施工/安装一级 | 双资质组合 |
| **纯土建施工** | 市政施工总承包二级及以上 | 单资质 |
| **机电设备改造** | 机电施工总承包一级 | 单资质 |
| **综合施工（含设备）** | 市政施工总承包一级 + 机电安装一级 | 双资质 |

### 2.3 施工业绩门槛对照

| 项目投资规模 | 业绩金额门槛 | 门槛比 |
|-------------|-------------|--------|
| ≥5000万元 | 2000-5000万元 | 40%-100% |
| 1000-5000万元 | 500-1500万元 | 30%-50% |
| <1000万元 | 按类型设定 | - |

### 2.4 施工类项目负责人

| 项目类型 | 建造师要求 | 专业 |
|----------|-----------|------|
| 市政工程 | 一级注册建造师 | 市政公用工程 |
| 机电安装 | 一级注册建造师 | 机电工程 |
| 建筑工程 | 一/二级注册建造师 | 建筑工程 |

## 三、设备类资质/条件对照

### 3.1 设备类投标人条件

| 要求项 | 常见配置 |
|--------|----------|
| 资质 | 制造商 或 授权代理商 |
| 安全生产许可证 | 如涉及安装工程则需 |
| 联合体 | 允许（安装单位牵头+设备供应商） |

### 3.2 设备类与设计/施工类的核心区别

| 维度 | 设计类 | 施工类 | 设备类 |
|------|--------|--------|--------|
| **资质体系** | 设计资质 | 施工总承包资质 | 供应/安装资质 |
| **项目负责人** | 注册工程师 | 注册建造师 | 项目经理（不一定要求注册） |
| **安全生产** | 一般不要求 | 必须有许可证 | 如含安装则需 |
| **业绩类型** | 设计业绩 | 施工业绩 | 设备供应/安装业绩 |
| **保证金** | 合同价2% | 合同价5-10% | 合同价5% |
| **预付款** | 0-10% | 10-30% | 30%（最高） |

---

## 四、联合体设置规律

| 项目类型 | 联合体策略 | 牵头人要求 |
|----------|-----------|-----------|
| **设计类** | 通常不接受 | - |
| **施工+O** | 接受，≤3家 | 主体施工单位（市政资质） |
| **设备采购安装** | 接受，≤2家 | 设备安装单位 |
| **纯施工** | 视项目而定 | 资质等级最高的单位 |

---

## 五、规模等级判定标准

### 污水处理厂项目

| 规模等级 | 日处理量 | 建筑工程等级 |
|----------|----------|-------------|
| 大型 | ≥10万m³/d | 大型 |
| 中型 | 2-10万m³/d | 中型 |
| 小型 | <2万m³/d | 小型 |

### 给排水管道项目

| 规模等级 | 管径 |
|----------|------|
| 大型 | ≥1.0m |
| 中型 | 0.5-1.0m |
| 小型 | <0.5m |

## 六、模板选择决策树

```
用户提供了项目信息
    ↓
判断项目类型
    ├── 关键词含「设计」→ 设计类
    ├── 关键词含「施工」「总承包」→ 施工类
    └── 关键词含「设备」「采购」「货物」→ 设备类
    ↓
判断项目规模
    ├── 大型 → 选择甲级/一级资质
    ├── 中型 → 选择乙级/二级资质
    └── 小型 → 选择丙级/三级资质
    ↓
匹配区域规律
    → 从 scoring-patterns.md 获取评分权重
    → 从 qualification-mapping.md 获取资质要求
    → 从 Step 3 结果获取具体区域数据
    ↓
填充模板变量
    → 输出完整模拟招标文件
```
