# 维度2：页面与前端代码对应表

## 页面：登录 (/login)

路径：app/(auth)/login/page.tsx

组件类型：客户端组件

### 组件：LoginForm

路径： app/(auth)/login/page.tsx （页面内组件）


界面位置：登录页居中卡片，包含邮箱、密码、记住我、登录按钮、注册/忘记密码链接

功能：收集登录凭据，调用 useAuthStore().login，成功后跳 callbackUrl

调用接口： POST /api/auth/login （auth-api.ts authLogin）

接口作用：接收 email + password → 校验凭据 → 返回 accessToken 并设置 httpOnly refresh cookie

### 组件：RememberMeCheckbox

路径： app/(auth)/login/page.tsx （页面内）

界面位置：密码输入框下方的复选框行

功能：勾选后凭据 base64 存入 localStorage（remember-credentials.ts），下次自动填充

调用接口：无

## 页面：注册 (/register)

路径：app/(auth)/register/page.tsx

组件类型：客户端组件

### 组件：RegisterForm

路径： app/(auth)/register/page.tsx （页面内）

界面位置：注册卡片：邮箱、邮箱验证码（60s 倒计时按钮）、密码、确认密码

功能：发送验证码 + 提交注册

调用接口：POST /api/auth/send-code（发送验证码）、POST /api/auth/register（authRegister）

接口作用：发送邮箱验证码；提交邮箱+密码+code → 创建用户 → 返回 accessToken

### 组件：SendCodeButton

路径： app/(auth)/register/page.tsx （页面内）

界面位置：邮箱输入框右侧的"发送验证码"按钮

功能：点击后调 sendCode，进入 60s 倒计时

调用接口：POST /api/auth/send-code

接口作用：向邮箱发送注册验证码，60s 冷却防刷
